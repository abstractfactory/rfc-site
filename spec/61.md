# Goal

To provide terminal access to the fundamental properties of Open Metadata.

# Architecture

```bash
$ om --name Marcus
$ om --name
Marcus
$ om --/address/street Blackwall Way
$ om --/address/street
Blackwall Way
$ om --/address
street=Blackwall Way
```
